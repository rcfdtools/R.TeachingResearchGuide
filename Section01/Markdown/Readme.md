## ¿Qué es el lenguaje Markdown?
Keywords: `Markdown` `HTML` `LaTeX`

Los repositorios de GitHub utilizan para la documentación el lenguaje Markdown que permite crear texto con formato, tablas, enlaces y utilizar elementos embebidos del lenguaje de hipertextos HTML y sin tener que definir estilos o tener conocimientos de programación. En este microcontenido encontrarás diferentes recursos para la escritura de e-Books y documentación wiki usando el lenguaje de marcas Markdown.
